# 국가를 당사자로 하는 소송에 관한 법률 시행령
> 대통령령 · 시행 1971-02-08 · 공포 1971-02-08 · 제정

### 제1조(목적)
이 영은 국가를 당사자로하는 소송에 관한 법률(이하"법"이라 한다)의 시행에 관하여 필요한 사항을 규정함을 목적으로 한다.

### 제2조(권한위임의 한계)
법무부장관은 법 제2조ㆍ제3조 및 제6조의 규정에 의한 권한을 다음 구분에 따라 검찰총장ㆍ고등검찰청검사장 및 지방검찰청검사장(이하 "각급 검찰청의 장"이라 한다)에게 위임한다. +----------+----------------------------------------------------------+ | 수 임 자 |             수    임    의         한    계              | +----------+----------------------------------------------------------+ | 검찰총장 |  대법원에 계속중인 법 제1조의 소송사건 및 행정소송사건   | |          |  (행정소송사건에 관하여는 법 제6조제2항의 규정에 의하여  | |          |  법무부장관이 수행하는 행정소송사건에 한한다).           | |          |                                                          | |고등검찰청|  대응하는 고등법원에 계속중인 법 제1조의 소송사건 및 행정| |  검사장  |  소송사건(행정소송사건에 관하여는 법 제6조제2항의 규정에 | |          |  의하여 법무부장관이 수행하는 행정소송사건에 한한다).    | |          |                                                          | |지방검찰청|  대응하는 지방법원 또는 그 지원에 관할권이 있는 법 제1조 | |  검사장  |  의 소송사건.                                            | +----------+----------------------------------------------------------+

### 제3조(수임사건에 대한 제한)
각급 검찰청의 장은 전조의 규정에 의하여 그 권한에 속한 소송사건중 소송물 가액이 100만원이상의 소송사건에 대하여 소의 제기 및 취하, 상소의 포기 및 취하, 화해ㆍ청구의 포기 및 인락의 소송행위를 하고자 할 때에는 미리 별표에 규정된 승인권자의 승인을 얻어야 한다.
