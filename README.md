# law_data

현행 시행 **법령**(법률·시행령·시행규칙)을 본문·링크·별표까지 Markdown 으로 담습니다.

`temporal_law` 파이프라인이 매일 수집해 자동으로 커밋·push 하는 **미러 저장소**입니다. (원본은 Postgres `lawdb` · 손으로 편집하지 마세요)

## 구조

```
법령/{패밀리}/{법률|시행령|시행규칙}/
  ├─ {법령명}.md      본문 (인라인 링크·이미지·⏳시행예정·⛔폐지예정)
  ├─ {법령명}.json    같은 내용의 구조화 데이터
  └─ 별표/ 별지/ 첨부파일/ 본문이미지/
```

같은 파일을 버전마다 덮어쓰며 커밋(커밋 날짜 = 시행일)하므로, **`git log` 가 곧 그 법의 개정 이력**입니다.
